# Divider

A versatile component for creating visual dividers with optional labels. Supports horizontal and vertical orientations with customizable colors and thickness. Perfect for separating content sections with descriptive labels or as simple divider lines.

## Installation

```bash
npx rn-shad-cli add divider
```

## Usage

### Basic Horizontal Divider

```tsx
import { Divider } from '@/components/ui/divider';
import { View, Text } from 'react-native';

<View>
  <Text>Section One</Text>
  <Divider />
  <Text>Section Two</Text>
</View>
```

### Divider with Label

```tsx
import { Divider } from '@/components/ui/divider';

<Divider label="OR" />
```

### Vertical Divider

```tsx
import { Divider } from '@/components/ui/divider';

<Divider orientation="vertical" />
```

### Custom Color

```tsx
import { Divider } from '@/components/ui/divider';

<Divider color="#FF0000" />
<Divider label="Continue" color="#3B82F6" />
```

### Custom Thickness

```tsx
import { Divider } from '@/components/ui/divider';

<Divider thickness={2} />
<Divider label="Separator" thickness={1.5} />
```

### Combined Props

```tsx
import { Divider } from '@/components/ui/divider';

<Divider
  label="Advanced Settings"
  orientation="horizontal"
  color="#9333EA"
  thickness={2}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | - | Optional text label to display in the center of horizontal divider |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Direction of the divider |
| `color` | `string` | `'#E4E4E7'` | Color of the divider line |
| `thickness` | `number` | `1` | Height (horizontal) or width (vertical) in pixels |
| `style` | `ViewStyle` | - | Additional style overrides |

## Features

- **Label Support**: Optional text labels centered on horizontal dividers
- **Flexible Orientation**: Support for both horizontal and vertical layouts
- **Customizable**: Full control over color and thickness
- **Semantic Dividers**: Ideal for separating form sections with labels
- **Lightweight**: Minimal component with no animations
- **Type-Safe**: Full TypeScript support

```tsx
import { Divider } from '@/components/ui/divider';

  const events = [
    { title: 'Order Placed', date: 'Nov 24, 2024', status: 'complete' },
    { title: 'Processing', date: 'Nov 24, 2024', status: 'complete' },

          <View style={{ flexDirection: 'row', gap: 12, alignItems: 'center' }}>
            <CheckCircle
              size={24}
              color={event.status === 'complete' ? '#16A34A' : '#A1A1AA'}

              <Text weight="semibold">{event.title}</Text>
              <Text color="muted" style={{ fontSize: 12 }}>

              </Text>
            </View>
          </View>

          {index < events.length - 1 && (
            <View style={{ marginLeft: 12, marginVertical: 12 }}>
              <Divider
                orientation="vertical"
                color={event.status === 'complete' ? '#16A34A' : '#E4E4E7'}
                thickness={2}
              />
            </View>
          )}
        </View>
      ))}
    </View>
  );
}
```

### Login/Signup Toggle

```tsx
import { Divider } from '@/components/ui/divider';
import { View, Text, Pressable } from 'react-native';
import { Input } from '@/components/ui/input';
import { Button } from '@/components/ui/button';

function LoginSignupForm() {
  const [isLogin, setIsLogin] = View.useState(true);

  return (
    <View style={{ padding: 16, gap: 20 }}>
      <View style={{ flexDirection: 'row', gap: 12 }}>
        <Pressable
          style={{ flex: 1, paddingVertical: 8 }}
          onPress={() => setIsLogin(true)}
        >
          <Text
            style={{
              textAlign: 'center',
              fontWeight: isLogin ? '600' : '400',
              color: isLogin ? '#18181B' : '#A1A1AA',
            }}
          >
            Sign In
          </Text>
        </Pressable>

        <Divider orientation="vertical" />

        <Pressable
          style={{ flex: 1, paddingVertical: 8 }}
          onPress={() => setIsLogin(false)}
        >
          <Text
            style={{
              textAlign: 'center',
              fontWeight: !isLogin ? '600' : '400',
              color: !isLogin ? '#18181B' : '#A1A1AA',
            }}
          >
            Sign Up
          </Text>
        </Pressable>
      </View>

      <Divider />

      <View style={{ gap: 12 }}>
        <Input
          placeholder={isLogin ? 'Email' : 'Full Name'}
        />
        <Input
          placeholder="Password"
          secureTextEntry
        />
        {!isLogin && (
          <Input
            placeholder="Confirm Password"
            secureTextEntry
          />
        )}
      </View>

      <Button>
        {isLogin ? 'Sign In' : 'Create Account'}
      </Button>
    </View>
  );
}
```

## Best Practices

1. **Label Text**: Keep labels short and descriptive (1-3 words):
   ```tsx
   <Divider label="OR" />
   <Divider label="Next Step" />
   ```

2. **Horizontal vs Vertical**:
   - Use `horizontal` for separating stacked sections
   - Use `vertical` for separating side-by-side content (requires explicit height)

3. **Semantic Labels**: Use descriptive labels that indicate what comes next:
   - "OR" for alternative options
   - "Next" for progression
   - Section names for grouped content

4. **Color Usage**: Match divider colors to content semantics:
   - Default/border color for neutral dividers
   - Success colors for completed steps
   - Warning colors for important sections

5. **Spacing**: Provide margin around dividers with labels:
   ```tsx
   <View style={{ marginVertical: 16, gap: 16 }}>
     <SectionOne />
     <Divider label="Section Two" />
     <SectionTwo />
   </View>
   ```

6. **Without Labels**: For simple separators without semantic meaning, use `Separator` component instead.

7. **Accessibility**: Ensure label text is concise and descriptive for screen readers.
